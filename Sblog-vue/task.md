# HTML & CSS 快速入门任务

欢迎来到 HTML 和 CSS 的世界！本任务专为小白设计，通过实践帮助你快速掌握网页开发的基础知识。

---

## 📋 任务概述

完成一个**个人名片网页**，包含你的基本信息、兴趣爱好展示和社交链接。

---

## 🎯 学习目标

- 掌握 HTML 基本结构和常用标签
- 理解 CSS 选择器和常用属性
- 学会使用 Flexbox 进行布局
- 实现简单的响应式设计

---

## 📝 任务清单

### 任务 1：搭建 HTML 基础结构

创建 `index.html` 文件，包含：

```
- DOCTYPE 声明
- html、head、body 标签
- meta 标签设置字符编码为 UTF-8
- title 标签设置网页标题
```

### 任务 2：编写 HTML 内容

在 body 中添加以下内容：

```
1. 头部区域（header）
   - h1 标签：你的名字
   - p 标签：一句个人简介

2. 关于我区域（section）
   - h2 标题："关于我"
   - p 标签：一段自我介绍（至少3句话）
   - ul 和 li 标签：列出3-5个兴趣爱好

3. 技能展示区域（section）
   - h2 标题："我的技能"
   - 使用 div 创建技能卡片（至少3个）
   - 每个卡片包含技能名称

4. 联系方式区域（footer）
   - h3 标题："联系我"
   - a 标签：添加2-3个社交链接
```

### 任务 3：添加 CSS 样式

创建 `style.css` 文件，设置以下样式：

```
1. 全局样式
   - 设置盒模型为 border-box
   - 设置默认字体和行高
   - 清除 body 默认边距

2. 头部样式
   - 背景色：深蓝色（#1a1a2e）
   - 文字颜色：白色
   - 居中对齐
   - 内边距：40px

3. 区域样式
   - 最大宽度：800px
   - 左右居中
   - 内边距：30px
   - 下外边距：20px
   - 圆角：10px
   - 阴影效果

4. 技能卡片样式
   - 使用 Flexbox 横向排列
   - 每个卡片背景色：浅蓝色（#e8f4f8）
   - 卡片内边距：15px
   - 卡片圆角：8px
   - 卡片间距：15px

5. 链接样式
   - 去掉下划线
   - 颜色：蓝色（#3498db）
   - 鼠标悬停时颜色变深
```

### 任务 4：进阶挑战（可选）

```
1. 添加头像图片（使用 img 标签）
2. 实现鼠标悬停时卡片放大效果
3. 添加页面背景渐变
4. 让技能卡片在移动端自动换行
```

---

## 💡 提示

- 使用 `<link>` 标签在 HTML 中引入 CSS 文件
- 使用 class 属性给元素添加类名，方便 CSS 选择
- 使用 Chrome 开发者工具（F12）可以实时查看效果
- 遇到问题可以搜索 MDN Web Docs 获取帮助
